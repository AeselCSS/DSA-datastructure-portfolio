## Big-O Time Complexity for SinglyLinkedList

| Operation        | Time Complexity |
|------------------|-----------------|
| add              | O(n)            |
| remove           | O(n)            |
| getFirst         | O(1)            |
| getLast          | O(n)            |
| getFirstNode     | O(1)            |
| getNextNode      | O(1)            |
| getLastNode      | O(n)            |
| getNodeWith      | O(n)            |
| removeFirstNode  | O(1)            |
| removeLastNode   | O(n)            |
| removeNode       | O(n)            |
| clear            | O(1)            |
| size             | O(1)            |
| dumpList         | O(n)            |
